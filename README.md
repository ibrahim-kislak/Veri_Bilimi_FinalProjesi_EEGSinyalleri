## Projeyi Çalıştırma Talimatı
Projeyi kendi bilgisayarınızda çalıştırmak için terminal veya komut satırını açıp `pip install pandas numpy matplotlib seaborn scikit-learn xgboost notebook` komutuyla gerekli tüm kütüphaneleri tek seferde yükleyin. Ardından, Kaggle platformundan indirilen `acquiredDataset.csv` veri dosyasını kod dosyasıyla aynı klasöre yerleştirin. Son olarak terminalde `jupyter notebook` komutunu çalıştırarak tarayıcıda açılan arayüzden proje notebook (`.ipynb`) dosyasını seçip tüm hücreleri sırasıyla çalıştırabilirsiniz.
